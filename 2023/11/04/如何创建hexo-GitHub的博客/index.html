如何创建hexo+GitHub的博客

第一次写博客，不知道要写什么，就来聊聊如何搭建一个自己的博客吧

第一步-下载git和node.js

首先，我们需要下载两个工具：git和node.js

git

到git官网上下载文件（下载地址https://git-scm.com/download）,安装选项全部默认，只不过最后一步添加路径时选择Use Git from the Windows Command Prompt，这样我们就可以直接在命令提示符里打开git了。

node.js

下载node.js（下载地址https://nodejs.cn/download/）,安装选项全部默认，一路点击Next。
最后安装好之后，按Win+R打开命令提示符，输入node -v和npm -v，如果出现版本号，那么就安装成功了。

第二步-注册GitHub账号并且创建仓库

注册github https://github.com/,然后在GitHub首页点create repository
注意选择public


注意仓库的取名格式：你的GitHub用户名.github.io(这将是以后的访问域名)
到这里代表我们Github账号以及仓库都已经创建完毕 可以把下面这段仓库的地址复制下来留着后面配置时会用到
（推荐使用ssh，https容易后期出现bug）

注：GitHub有可能加载不出来，多试几次或者在微软商店里下个watt toolkit启用加速就可以了。

第三步- 安装hexo

前面git和nodejs安装好后，就可以安装hexo了，你可以先创建一个文件夹MyBlog，用来存放自己的博客文件
在该目录下右键点击Git Bash Here，输入npm install -g hexo-cli安装Hexo。
安装完后输入hexo -v验证是否安装成功。
至此hexo就安装完了。
接下来初始化一下hexo,即初始化我们的网站，在git bach here 里输入hexo init初始化文件夹
接着输入npm install安装必备的组件。
新建完成后，指定文件夹Hexo目录下有：

node_modules
public
scaffolds
source
themes
_config.yml
这样本地的网站配置也弄好啦，输入hexo g生成静态网页，然后输入hexo s打开本地服务器，按ctrl+c关闭本地服务器。

生成SSH添加到GitHub

生成SSH添加到GitHub，连接Github与本地。

右键打开git bash，然后输入下面命令：

git config –global user.name “yourname”
git config –global user.email “youremail”

注意：第一次使用git后需要将用户名和邮箱进行初始化
这里的yourname输入你的GitHub用户名，youremail输入你GitHub的邮箱。这样GitHub才能知道你是不是对应它的账户。
注：可以用以下两条，检查一下你有没有输对
git config user.name
git config user.email

然后创建SSH

在git bash hrre 上输入ssh-keygen -t rsa -C “youremail”生成秘钥
接着去查看C盘目录(C:\用户\用户名目录下)，然后在记事本中打开id_rsa.put文件并复制生成的key

把ssh部署到你的GitHub上

打开github，在头像下面点击settings，再点击SSH and GPG keys，新建一个SSH，名字随便取一个都可以，把你的id_rsa.pub里面的信息复制进去。
在git bash here 输入ssh -T git@github.com，如果出现你的用户名，那就成功了。


将hexo部署到你的GitHub上

这一步，我们就可以将hexo和GitHub关联起来，也就是将hexo生成的文章部署到GitHub上，打开博客根目录下的_config.yml文件，

修改最后一行的配置：
deploy:
type: git
repository: https://github.com/你的GitHub用户名/你的GitHub用户名.github.io.git
branch: main
这个时候需要先安装deploy-git ，也就是部署的命令,这样你才能用命令部署到GitHub。
在git bash 上输入npm install hexo-deployer-git –save
然后输入
hexo clean
hexo generate
hexo deploy
注：hexo generate顾名思义，生成静态文章，可以用 hexo g缩写 ，hexo deploy部署文章，可以用hexo d缩写。

新建文章(不新建的话也会有一篇自带的Hello World文章)

输入命令：hexo new “文章名称”

然后根据返回的地址即可找到创建的文件位置，即可使用编辑器打开编辑（推荐使用vscode或者typora）

预览

输入命令（blog目录下使用git）：hexo s（如上）

确认无误后先生成一遍文件

输入命令（blog目录下使用git）：hexo g

之后即可部署到GitHub

输入命令（blog目录下使用git）：hexo d
注：这步记得关加速器，我搭博客时忘了关加速器结果在这一步上卡了好久，结果把加速器一关就可以了:-)

做完这些，你的博客就搭好了！！！！！
